# ChromaDB 벡터 DB 초기화 가이드

## 📋 벡터 DB 초기화 방법

### 방법 1: 폴더 삭제 (권장)

```bash
# 프로젝트 루트에서 실행
rm -rf data/chroma/
```

**주의**: 
- 서버가 실행 중이면 먼저 서버를 종료하세요 (Ctrl+C)
- 삭제 후 서버 재시작 시 자동으로 새 컬렉션이 생성됩니다

### 방법 2: 단계별

```bash
# 1. 서버 종료 (실행 중인 경우)
# Ctrl+C

# 2. 벡터 DB 폴더 삭제
cd /Users/mac/Desktop/hateslop/Yeop-Gang
rm -rf data/chroma/

# 3. 서버 재시작
cd server
uvicorn main:app --reload
```

## 📍 벡터 DB 경로

- **기본 경로**: `data/chroma/`
- **설정**: `.env` 파일의 `CHROMA_DB_PATH` 환경 변수 (선택사항)

## ⚠️ 주의사항

1. **모든 데이터 삭제**: 폴더 삭제 시 모든 저장된 강의 데이터가 사라집니다
2. **재업로드 필요**: 초기화 후 강의 파일을 다시 업로드해야 합니다
3. **서버 종료**: 서버가 실행 중이면 먼저 종료하세요

## ✅ 초기화 후 다음 단계

1. 벡터 DB 초기화 (폴더 삭제)
2. 서버 재시작
3. 올바른 강의 파일 재업로드
4. 새로운 ID 생성 로직으로 모든 세그먼트가 개별 저장됨
